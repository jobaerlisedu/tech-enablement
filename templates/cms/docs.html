1. Getting Started with the CMS

This CMS enables editors to publish research blogs and manage structured learning courses connected directly to Google Firebase. Because all content is stored as document records in Firestore, updates are instantly synchronized to visitors on the frontend.

3. Managing Courses & Tutorial Lessons

To construct a structured course:

  1. First, register a Category and an Author/Instructor.
  2. Create the Course in the Course Management panel. Set the Display Sequence Order (e.g., 1 for first, 2 for second) to order cards.
  3. Create Lessons under Tutorial Management, linking them to your Course. Give each lesson an Order Number (e.g., 1 for introduction, 2 for environment setup) so students can click Next/Prev to walk through the course in the correct sequence.

4. File Uploads & Storage Bucket

When creating authors, blogs, or courses, you can select an image file to upload directly to Google Firebase Storage. The system automatically handles name conflicts, uploads the file, sets its permissions to public read, and retrieves the URL to store inside your Firestore document records.
